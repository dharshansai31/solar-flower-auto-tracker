# Docs

Place the project report here, e.g.:

- `Auto_Solar_Pilot_Report.pdf` — full mini-project report (background, methodology,
  hardware/software details, calibration, field testing, results, references)

If you have a PDF or DOCX version of the report, copy it into this folder and
make sure the filename matches the link used in the main README's
"References" / "Results and Conclusion" sections.
